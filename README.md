# indic_transliteration

Transliterate English to Indian language or vice-versa

## Setup
```
#Clone repository

#create environment
virtualenv -p python3 .
source bin/activate
pip install -r requirements.txt
```

## Command line usage
```
python transliteration.py --filename sample.xlsx --lang hi
```
